Algunas categorías de los algoritmos son
- **Search**: Algoritmo para buscar un elemento dentro de una estructura de datos.
- **Sort**: Algoritmo para ordenar elemento en un cierto orden.
- **Insert**: Insertar un item dentro de una estructura de datos.
- **Update**: Actualizar un item dentro de una estructura de datos.
- **Delete**: Algoritmo para eliminar un item dentro de una estructura de datos.
#### Características de un algoritmo
- **Preciso**: Un algoritmo debe de ser claro y preciso. Cada uno de sus pasos, y sus entradas / salidas tienen que ser claros y tienen que llevar la solución a un único significado.
- **Input**: Tiene que estar bien definidos los parámetros de entrada.
- **Output**
- **Finito**
- **Viabilidad**
- **Independiente**
